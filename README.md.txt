# Street Flags App

Dashboard de gestión, ventas y control de inventario para uso interno. Diseñado específicamente para formato móvil.

## Características Principales
- **Dashboard:** Panel de métricas mensuales.
- **Ventas:** Registro y proyecciones basadas en historial.
- **Inventario y Recetas:** Control de stock y costeo de platos en tiempo real.
- **Generador de Pedidos:** Calculadora automática de compras y exportación a WhatsApp.
- **PWA (Progressive Web App):** Instalable directamente en dispositivos iOS y Android.
